Codemap Update Report - 2026-04-30
================================

Major architectural shifts detected and documented:

1. Distributed State Management:
   - Redis integration for cross-instance presence and cache invalidation.
   - Deterministic session takeover logic for Free-tier users.

2. Monetization & Retention:
   - "Pro Plan" vs "Free Plan" infrastructure.
   - Plan-aware message history scrubbing and connection limits.
   - Account downgrade background jobs via Agenda.

3. Authentication & Security:
   - New flows for password reset and email verification.
   - Failed-through protection in rate limiting.
   - Idempotency checks for real-time messaging.

4. Data Model Flexibility:
   - Transition to `participants` array in Chats to support future group chat features.
   - Bio and email verification status added to User profiles.

5. Toolchain Modernization:
   - Migration to Vite Plus (vp) and Bun.
   - Vitest for unified testing.
